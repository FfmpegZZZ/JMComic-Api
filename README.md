## 使用方法
> **Warning:** api获取的是绝对路径，在docker中使用时请注意!
1. **下载源码包并解压**，进入项目根目录
    
2. **创建虚拟环境**：
     ```bash
     python -m venv .venv
     ```

3. **激活虚拟环境**：
   - 在 Windows 上：
     ```bash
     .\.venv\Scripts\activate
     ```
   - 在 macOS 或 Linux 上：
     ```bash
     source .venv/bin/activate
     ```

4. **安装依赖**：
     ```bash
     pip install -r requirements.txt
     ```
